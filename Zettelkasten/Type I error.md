Also called $\displaystyle \large \alpha$ error.

The error when we *reject* the [[Null Hypothesis]] when it is true.

| Null Hypothesis | Actual Situation = *TRUE* |
| --------------- | ----------------------- |
| Fail to reject  | Correct decision        |
| Reject          | **Type I Error**            |
