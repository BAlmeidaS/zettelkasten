[[Random variable|Random variables]] can be defined in the following tree:

- [[nominal variable]]
- [[ordinal variable]] 
- [[interval variable]] 
- [[ratio variable]]

This definition was proposed by *Stanley Smith Stevens*, so are commonly called *stevens' scale of measurement*
