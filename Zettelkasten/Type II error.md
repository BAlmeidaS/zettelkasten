Also called $\displaystyle \large \beta$ error.

The error when we *accept* a [[Null Hypothesis]] when it is false.

| Null Hypothesis | Actual Situation = *FALSE* |
| --------------- | ------------------------ |
| Fail to Reject  | **Type II Error**            |
| Reject          | Correct decision         |
