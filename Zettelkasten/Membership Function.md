A function that defines how much an element *belongs* to a [[Fuzzy Sets|Fuzzy Set]]. It varies from 0 to 1.
$$\displaystyle \Huge \mu_a(x):X\rightarrow[0,1]$$


It can be represented as a function, or as a table.
